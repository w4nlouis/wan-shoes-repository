# Wan Shoes Inventory Management System

## Project Overview
Internal inventory system for Wan Shoes shop. Admin and employees only (no customer frontend).

## Technologies
- PHP + MySQL
- HTML/CSS
- JavaScript (form validation, password strength)

## Features
- Role-based login (Admin, Clerk, Stock Manager)
- Product management (CRUD operations)
- Employee management (Admin only)
- Sales recording (Clerk)
- Stock management (Stock Manager)
- Sales reports with revenue totals

## Database Structure
- admins
- employees  
- products
- sales
- stock_records

## Installation
1. Copy folder to `C:\xampp\htdocs\wan_shoes`
2. Import `wan_shoes_db.sql` to phpMyAdmin
3. Access at `http://localhost/wan_shoes/login.php`
